# **GNU STOW - .dotfiles**

## Install stow

`sudo pacman -S stow`

## Clone repozytory .dotfiles

`cd ~/`

`git clone --depth=1 https://github.com/trebuhw/.dotfiles.git`

## Usage stow

`cd ~/.dotfiles/`

`stow .` # all ln -s link .dotfiles to ~/.config

or

np: `stow kitty/` (ln -s link .dotfiles to ~/.config/kitty)
